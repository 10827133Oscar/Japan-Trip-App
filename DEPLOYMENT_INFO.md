## 📝 注意事項

### Google Maps API 限制
- 確保 API Key 已啟用「Maps JavaScript API」（網頁版）
- 確保 API Key 已啟用「Maps SDK for Android」（APK）
- 每月免費額度：$200（約 28,000 次地圖載入）
- 如果超過免費額度，Google 會發信通知

### Firebase 限制
- Spark 免費方案限制：
  - 儲存：1 GB
  - 下載：10 GB/月
  - 文件讀取：50,000 次/天
  - 文件寫入：20,000 次/天
- 目前用量遠低於限制，不用擔心

### Vercel 限制
- Hobby 免費方案：
  - 無限網站
  - 100 GB 流量/月
  - 自動 HTTPS
  - 自動 CDN
- 目前用量遠低於限制

## 🐛 故障排除

### 網頁版看不到地圖

1. 檢查瀏覽器控制台（F12）
2. 確認 Google Maps API Key 正確
3. 確認 API Key 已啟用「Maps JavaScript API」
4. 清除瀏覽器快取重新載入

### APK 無法連接 Firebase

1. 檢查網路連接
2. 確認 Firebase 配置正確
3. 重新安裝 APK
4. 查看 BUILD_INFO.md 的故障排除章節

### 資料不同步

1. 確認所有裝置使用相同的計畫 ID
2. 檢查網路連接
3. 嘗試下拉刷新
4. 重新登入計畫
