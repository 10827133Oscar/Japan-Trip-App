## 1️⃣ 用戶註冊與登入

### 1.1 首次使用
- [ ] 打開 App/網頁
- [ ] 顯示歡迎畫面
- [ ] 輸入暱稱（測試：1-10 個字）
- [ ] 選擇代表色（測試所有 8 種顏色）
- [ ] 點擊「開始使用」
- [ ] 成功進入主頁面

### 1.2 暱稱驗證
- [ ] 輸入空暱稱 → 應顯示錯誤提示
- [ ] 輸入超過 10 個字 → 應顯示錯誤提示
- [ ] 輸入有效暱稱 → 成功

### 1.3 顏色選擇
- [ ] 點擊不同顏色 → 顏色正確切換
- [ ] 已選擇的顏色有視覺反饋（✓ 標記）

## 2️⃣ 計畫管理

### 2.1 創建計畫
- [ ] 點擊「創建計畫」按鈕
- [ ] 輸入計畫名稱（例如：「東京家庭旅遊」）
- [ ] 輸入目的地（例如：「東京」）
- [ ] 設定計畫密碼（例如：「1234」）
- [ ] 點擊「創建」
- [ ] 顯示計畫 ID（6 位數字）
- [ ] 計畫 ID 可點擊複製
- [ ] 成功創建後自動進入計畫

### 2.2 查看計畫列表
- [ ] 主頁面顯示所有已加入的計畫
- [ ] 顯示計畫名稱、目的地、參與者數量
- [ ] 顯示當前選中的計畫（有視覺標記）

### 2.3 加入計畫
- [ ] 點擊「加入計畫」按鈕
- [ ] 輸入計畫 ID（6 位數字）
- [ ] 輸入計畫密碼
- [ ] 點擊「加入」
- [ ] 成功加入後顯示計畫詳情
- [ ] 計畫列表更新，顯示新加入的計畫

### 2.4 計畫密碼驗證
- [ ] 輸入錯誤的計畫 ID → 顯示錯誤提示
- [ ] 輸入錯誤的密碼 → 顯示錯誤提示
- [ ] 輸入正確的 ID 和密碼 → 成功加入

### 2.5 切換計畫
- [ ] 點擊不同的計畫卡片
- [ ] 計畫正確切換
- [ ] 景點列表更新為當前計畫的景點
- [ ] 地圖更新為當前計畫的景點

### 2.6 查看計畫詳情
- [ ] 點擊計畫卡片上的「詳情」或類似按鈕
- [ ] 顯示計畫完整資訊：
  - [ ] 計畫名稱
  - [ ] 目的地
  - [ ] 開始日期
  - [ ] 結束日期
  - [ ] 參與者列表（顯示暱稱和顏色）
  - [ ] 計畫 ID

### 2.7 退出計畫
- [ ] 點擊「退出計畫」按鈕
- [ ] 顯示確認對話框
- [ ] 確認退出
- [ ] 成功退出計畫
- [ ] 計畫從列表中移除
- [ ] 自動切換到其他計畫（如果有的話）

---

## 3️⃣ 景點管理

### 3.1 新增景點（從景點頁面）
- [ ] 切換到「景點」頁面
- [ ] 點擊右下角「+」按鈕
- [ ] 填寫景點資訊：
  - [ ] 景點名稱（必填）
  - [ ] 地址（必填）
  - [ ] 類型（下拉選單）
  - [ ] 第幾天（數字輸入）
  - [ ] 備註（可選）
- [ ] 點擊「新增景點」
- [ ] 成功新增後返回景點列表
- [ ] 新景點顯示在列表中
- [ ] 景點顯示正確的顏色（根據創建者）

### 3.2 新增景點（從地圖頁面）
- [ ] 切換到「地圖」頁面
- [ ] 點擊「新增景點」按鈕
- [ ] 進入新增模式（地圖顯示提示）
- [ ] 點擊地圖上的位置
- [ ] 自動填入地址（Geocoding）
- [ ] 填寫其他資訊
- [ ] 點擊「新增景點」
- [ ] 成功新增後地圖顯示新標記

### 3.3 查看景點列表
- [ ] 景點列表正確顯示所有景點
- [ ] 景點按天數分組或排序
- [ ] 顯示景點名稱、地址、類型
- [ ] 顯示創建者的顏色標記
- [ ] 顯示第幾天資訊

### 3.4 篩選景點
- [ ] 點擊「全部」→ 顯示所有景點
- [ ] 點擊「第 1 天」→ 只顯示第 1 天的景點
- [ ] 點擊「第 2 天」→ 只顯示第 2 天的景點
- [ ] 篩選後列表正確更新
- [ ] 地圖同步更新（只顯示篩選的景點）

### 3.5 編輯景點
- [ ] 點擊景點卡片或「編輯」按鈕
- [ ] 進入景點詳情頁面
- [ ] 修改景點資訊：
  - [ ] 名稱
  - [ ] 地址
  - [ ] 類型
  - [ ] 第幾天
  - [ ] 備註
- [ ] 點擊「儲存」
- [ ] 成功更新後返回列表
- [ ] 列表顯示更新後的資訊
- [ ] 地圖標記位置更新（如果地址改變）

### 3.6 刪除景點
- [ ] 在景點詳情頁面點擊「刪除」
- [ ] 顯示確認對話框
- [ ] 確認刪除
- [ ] 成功刪除後返回列表
- [ ] 景點從列表中移除
- [ ] 地圖標記移除

### 3.7 查看景點詳情
- [ ] 點擊景點卡片
- [ ] 顯示完整景點資訊：
  - [ ] 名稱
  - [ ] 地址
  - [ ] 類型
  - [ ] 第幾天
  - [ ] 備註
  - [ ] 創建者資訊
- [ ] 顯示「帶我去」按鈕
- [ ] 顯示「編輯」按鈕
- [ ] 顯示「刪除」按鈕

### 3.8 帶我去功能
- [ ] 在景點詳情頁面點擊「帶我去」
- [ ] 跳轉到地圖頁面
- [ ] 地圖自動定位到該景點
- [ ] 景點標記高亮顯示

## 4️⃣ 地圖功能

### 4.1 地圖顯示
- [ ] 切換到「地圖」頁面
- [ ] 地圖正確載入（無錯誤）
- [ ] 地圖顯示當前計畫的所有景點標記
- [ ] 標記顏色對應景點類型
- [ ] 標記位置準確

### 4.2 地圖互動
- [ ] 可以拖動地圖
- [ ] 可以縮放地圖（雙指縮放或滾輪）
- [ ] 點擊標記顯示景點資訊（名稱、地址）
- [ ] 點擊標記後可以查看詳情或導航

### 4.3 顯示全部景點
- [ ] 點擊「顯示全部」按鈕
- [ ] 地圖自動調整視角，顯示所有景點
- [ ] 所有標記都在可見範圍內

### 4.4 從地圖新增景點
- [ ] 點擊「新增景點」按鈕
- [ ] 進入新增模式
- [ ] 地圖顯示提示「請點擊地圖選擇位置」
- [ ] 點擊地圖上的位置
- [ ] 自動填入地址
- [ ] 填寫其他資訊
- [ ] 成功新增後地圖顯示新標記

### 4.5 路線規劃
- [ ] 確保有至少 2 個景點
- [ ] 點擊「規劃路線」按鈕
- [ ] 顯示路線規劃結果
- [ ] 地圖顯示路線（如果支援）
- [ ] 顯示路線距離和時間

### 4.6 用戶位置
- [ ] 請求位置權限（首次使用）
- [ ] 地圖顯示當前位置（藍點）
- [ ] 位置更新正確

### 5.4 顏色區分
