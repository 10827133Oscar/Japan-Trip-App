## 📝 設計步驟

### Step 1: 使用 Canva 快速設計

1. **註冊/登入 Canva**
   - 前往 https://www.canva.com/
   - 使用 Google 帳號快速登入

2. **創建設計**
   - 點擊「創建設計」
   - 選擇「自訂尺寸」
   - 輸入 1024 x 1024 px
   - 點擊「創建新設計」

3. **設計背景**
   - 點擊左側「元素」
   - 搜尋「正方形」
   - 拖放到畫布
   - 調整大小至滿版
   - 點擊顏色，輸入 `#007AFF`

4. **添加圖示元素**
   - 點擊左側「元素」
   - 搜尋「地圖標記」或「富士山」
   - 選擇簡潔的圖示
   - 拖放到中央
   - 調整大小（建議佔畫布 60-70%）
   - 設定顏色為白色

5. **添加日本元素（可選）**
   - 搜尋「櫻花」或「日本」
   - 添加小元素作為裝飾
   - 不要過於複雜

6. **下載**
   - 點擊右上角「下載」
   - 選擇「PNG」格式
   - 下載並重命名為 `icon.png`

### Step 2: 創建 Android 自適應圖示

Android 自適應圖示需要考慮安全區域：

```
┌────────────────────────┐
│   20% 邊距             │
│  ┌──────────────┐      │
│  │              │      │
│  │  主要圖示    │ 60%  │
│  │  放置區域    │      │
│  │              │      │
│  └──────────────┘      │
│   20% 邊距             │
└────────────────────────┘
```

**重要:** 主要元素需要在中央 60% 區域內，避免被裁切。

**步驟:**
1. 複製 `icon.png`
2. 在 Canva 中打開
3. 確保主要元素縮小至 60% 區域
4. 背景可以延伸到全畫布
5. 下載並重命名為 `adaptive-icon.png`

### Step 3: 創建啟動畫面

**尺寸:** 1284 x 2778 px (iPhone 14 Pro Max)

**設計建議:**
- 使用相同的藍色背景 `#007AFF`
- 中央放置應用圖示（稍大一些）
- 下方可添加應用名稱「日本旅遊助手」
- 保持簡潔

**步驟:**
1. 在 Canva 創建 1284 x 2778 的設計
2. 設定藍色背景
3. 放置圖示於中央
4. 下載並重命名為 `splash-icon.png`

### Step 4: 創建網頁圖示

**尺寸:** 48 x 48 px

**步驟:**
1. 在 Canva 縮小 `icon.png` 至 48x48
2. 或使用線上工具自動轉換
3. 下載並重命名為 `favicon.png`

## 🎨 設計參考

### 靈感來源

**類似應用:**
- Google Maps - 簡潔的地圖標記
- TripAdvisor - 貓頭鷹 + 旅遊元素
- Airbnb - 簡單的符號

**日本元素:**
- 🗻 富士山輪廓
- 🌸 櫻花
- ⛩️ 鳥居
- 🗾 日本地圖
- 🏯 城堡
- 📍 地圖標記

### 設計原則

1. **簡潔至上**
   - 在小圖示上仍能清楚辨識
   - 避免太多細節

2. **品牌一致**
   - 使用統一的藍色
   - 風格保持一致

3. **可辨識性**
   - 一眼就能知道是旅遊相關
   - 與其他 App 有區別

4. **文化適配**
   - 使用日本元素但不刻板
   - 現代簡約風格

## 🚀 完成後

圖示設計完成並放入 `assets/` 目錄後：

```bash
# 檢查圖示
ls -la assets/

# 應該看到：
# icon.png
# adaptive-icon.png
# splash-icon.png
# favicon.png
```

然後可以開始建構應用：

```bash
# 安裝 EAS CLI（如果還沒安裝）
npm install -g eas-cli

# 建構預覽版本
eas build --profile preview --platform all
```
